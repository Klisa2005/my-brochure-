# 📖 Full-Screen Slide Website (QR-Based Brochure)

## 🚀 Project Overview
This project is a **full-screen image-based website** that works like a digital brochure.

- Users scan a **QR code**
- Website opens in browser
- Images are displayed **slide-by-slide (full screen)**
- Users can **scroll to navigate**
- Each page can be **downloaded or printed**

---

## ✨ Features

### 🖼️ Full-Screen Slides
- Each image covers the entire screen (100vh)
- No unwanted background
- Clean and modern UI

### 📜 Scroll-Based Navigation
- Vertical scrolling moves to next page
- Uses **scroll snap** for smooth slide transitions

### ⬅️➡️ Navigation Buttons
- Previous / Next buttons
- Smooth hover and click animations

### ⬇️ Download Option
- Download the currently visible image

### 🎬 Animations
- Button hover (lift + glow)
- Click press effect
- Smooth image transitions (fade in/out)

### 🖨️ Print Support
- Each slide prints on a separate page
- Maintains original layout and format

### 📱 Mobile Friendly
- Works on all screen sizes
- Optimized for mobile scrolling
- (Optional: swipe gestures)
